# To-Do App ✅

A simple and responsive To-Do List web app built using **HTML, CSS, and JavaScript**.

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Responsive design for all screen sizes

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
